## x-py
> Experimental and development Python packages for Leraniode Projects

![python](https://img.shields.io/badge/python-3.10%2B-blue)
![license](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-experimental-orange)

**x-py** contains the official experimental, cutting-edge Python libraries, modules, and tools developed under Leraniode.

These packages are:
- Lightweight and modular
- Focused on rapid experimentation
- Designed for prototyping new ideas, workflows, and systems
